# Predictive Biomarker Discovery via Machine Learning

## 📌 Overview
This repository contains a Machine Learning pipeline designed to extract, predict, and rank the Top 50 high-impact genetic biomarkers from differential gene expression data. Built primarily for oncology research (such as breast cancer biomarker identification), the script leverages a **Random Forest Regressor** to evaluate statistical gene metrics and classify genes as potential **Cancer Drivers** (Upregulated) or **Tumor Suppressors** (Downregulated).

This code is optimized to produce publication-ready datasets and high-resolution figures for academic journals.

## 🔬 Methodology
The pipeline executes the following workflow:
1.  **Data Ingestion & Preprocessing:** Loads clean, stacked gene expression data (`AveExpr`, `t`, `P.Value`, `adj.P.Val`, `B`). Missing values are handled via mean imputation, and features are standardized using `StandardScaler` to ensure uniform influence.
2.  **Model Training:** A `RandomForestRegressor` (50 estimators) is trained on the scaled statistical features to predict the biological magnitude of change (`logFC`).
3.  **Biomarker Scoring:** Calculates an **ML Impact Score** based on the absolute predicted `logFC`. Extreme up-regulation and down-regulation are treated as equally critical biological markers.
4.  **Biological Classification:** Genes are ranked by their Impact Score and classified based on their actual `logFC` direction:
    * *logFC > 0:* Upregulated (Cancer Driver)
    * *logFC < 0:* Downregulated (Tumor Suppressor)
5.  **Visualization:** Generates a high-quality, color-coded horizontal bar chart (Figure 2) detailing the Top 50 predictive biomarkers.

## 📂 Repository Contents
